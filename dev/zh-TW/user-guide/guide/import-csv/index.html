    匯入 CSV¶

    AssetFlow 的 CSV 匯入功能讓你從試算表匯出的檔案中批次匯入資產與現金流。這是利用現有投資組合資料快速上手的最佳方式，也方便你批次更新持倉。

    開啟匯入工具¶

    在側邊欄中前往工具 > 匯入 CSV，或按 Cmd+I。

    在匯入檢視的頂部，切換資產或現金流來選擇你要匯入的類型。

    匯入資產¶

    步驟 1：選取檔案¶

    將 CSV 檔案拖放到匯入區域，或點選瀏覽從 Finder 中選取。

    你的檔案至少應包含以下欄位：

    欄位 必填 說明
    Asset Name 是 資產名稱
    Market Value 是 目前的市場價值
    Platform 否 持有此資產的平台
    Currency 否 資產的幣別代碼（例如 USD、TWD）

    步驟 2：設定¶

    匯入前，你可以微調資料的匯入方式：

    • 快照日期：選擇此快照代表的日期。
    • 平台：可選擇指定一個平台。如有需要，可以直接建立新平台。勾選套用至所有列會將其套用到每一項匯入的資產；不勾選則只會填入 CSV 中沒有平台資訊的列。
    • 類別：可選擇指定一個類別。同樣可以直接建立新類別。套用至所有列的運作方式相同，勾選會覆蓋所有列，不勾選只填入空白的列。
    • 複製帶入：開啟此選項可帶入不在此 CSV 中的其他平台資產。這在你只從某一家券商匯入，但想保留其他平台資產在同一筆快照中時很有用。
      • 選擇要從哪些平台複製帶入。
      • 每個選項會顯示資產數量與來源快照日期。

    步驟 3：檢視與匯入¶

    預覽表格顯示所有解析後的列及其驗證狀態：

    • 錯誤（紅色）：這些問題會阻止匯入，必須修正。常見原因包括缺少必填欄位、重複的資產，或無法解析的值。
    • 警告（黃色）：允許匯入但標記潛在問題，例如數值為零或無法辨識的幣別代碼。

    你可以在匯入前直接從預覽中移除有問題的列。

    確認一切無誤後，點選匯入。

    Warning

    如果所選日期已有快照存在，匯入會更新該快照，而非建立新的快照。

    匯入現金流¶

    現金流匯入的流程與資產類似。你的 CSV 應包含：

    欄位 必填 說明
    Description 是 現金流的標籤（例如「每月定期投入」）
    Amount 是 正數代表存入，負數代表提領
    Currency 否 幣別代碼

    現金流會匯入你在設定時選擇的特定快照中。
