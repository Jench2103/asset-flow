    CSV 格式¶

    AssetFlow 匯入工具接受的 CSV 檔案格式參考。

    資產 CSV¶

    必填欄位¶

    欄位 說明
    Asset Name 資產名稱（例如「AAPL」、「Vanguard Total Stock Market ETF」）
    Market Value 目前的市場價值，以數字表示（例如 15000、1234.56）

    選填欄位¶

    欄位 說明
    Platform 持有此資產的平台或券商
    Currency 三碼幣別代碼（例如 USD、EUR、TWD）。省略時預設為你的主要幣別。

    範例¶

    Asset Name,Market Value,Platform,Currency
AAPL,15000,Interactive Brokers,USD
VTI,28000,Interactive Brokers,USD
Bitcoin,5000,Coinbase,USD
台積電,120000,永豐金證券,TWD

    現金流 CSV¶

    必填欄位¶

    欄位 說明
    Description 現金流的標籤（例如「每月定期投入」、「股息再投入」）
    Amount 金額，以數字表示。正數代表存入，負數代表提領。

    選填欄位¶

    欄位 說明
    Currency 三碼幣別代碼。省略時預設為你的主要幣別。

    範例¶

    Description,Amount,Currency
Monthly savings,5000,USD
Emergency withdrawal,-2000,USD
年終獎金投入,100000,TWD

    常見問題¶

    問題 原因 解決方法
    「缺少必填欄位」 CSV 沒有 Asset Name 或 Market Value 標題 檢查欄位標題是否完全一致
    「重複的資產」 同一個資產名稱 + 平台的組合出現了兩次 移除重複的列或變更平台
    「無法解析的數值」 Market Value 包含非數字字元 移除貨幣符號、數字中的逗號
    「不支援的幣別」 幣別代碼無法辨識 使用標準 ISO 4217 代碼（例如 USD、EUR、TWD）

    Tip

    將試算表匯出為 UTF-8 CSV，以確保特殊字元（例如中文）能被正確處理。
