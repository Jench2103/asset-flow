# Keyboard Shortcuts

Quick reference for keyboard shortcuts in AssetFlow.

| Shortcut      | Action              |
| ------------- | ------------------- |
| ++cmd+n++     | Create new snapshot |
| ++cmd+i++     | Open CSV import     |
| ++cmd+comma++ | Open Settings       |

!!! note

    More keyboard shortcuts may be added in future updates.
