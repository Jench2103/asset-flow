# 開始使用

歡迎使用 AssetFlow！這個頁面會帶你了解系統需求、安裝方式，以及應用程式的快速導覽，讓你知道每個功能在哪裡。

---

## 系統需求

| 需求         | 說明                                           |
| ------------ | ---------------------------------------------- |
| **作業系統** | macOS 15.0（Sequoia）或更新版本                |
| **架構**     | Apple Silicon 或 Intel Mac                     |
| **磁碟空間** | 極小，AssetFlow 使用輕量級資料庫在本機儲存資料 |
| **網路**     | 選用，僅用於取得匯率                           |

---

## 安裝

1. 前往 GitHub 上的 [Releases](https://github.com/Jench2103/asset-flow/releases) 頁面，下載最新的 **AssetFlow-x.y.z.zip**。
1. 解壓縮檔案，將 **AssetFlow.app** 移至**應用程式**資料夾。
1. 從**應用程式**或 Spotlight（++cmd+space++ 輸入「AssetFlow」）啟動 AssetFlow。

### 略過 Gatekeeper 警告

由於 AssetFlow 未使用 Apple 開發者憑證簽署，macOS 會在首次啟動時阻擋它。這是正常現象，以下是允許開啟的步驟：

1. 雙擊 **AssetFlow.app**，你會看到一個無法開啟應用程式的警告。
1. 打開**系統設定** → **隱私權與安全性**。
1. 向下捲動到**安全性**區段，你會看到一段訊息：*「AssetFlow 因為不是來自已識別的開發者而遭到阻擋。」*
1. 點選**仍要打開**，然後使用 Touch ID 或密碼進行驗證。
1. 在確認對話框中，點選**打開**。

之後每次開啟都不會再出現警告。

!!! note

    這是一次性的步驟。一旦你透過 Gatekeeper 允許了 AssetFlow，macOS 會記住你的選擇，不會再次詢問。

## 首次啟動

第一次開啟 AssetFlow 時，你會看到一個空白的儀表板，以及引導你開始使用的提示。不需要建立帳號，也不需要登入，所有功能都在你的 Mac 上本機運作。

![空白儀表板](../../../assets/images/dashboard-empty-state.png)

從這裡你可以：

- **建立你的第一筆快照**，開始追蹤投資組合
- **從 CSV 檔案匯入資料**，如果你已經有試算表中的持倉資料

---

## 應用程式總覽

AssetFlow 以左側的側邊欄為核心，讓你快速存取應用程式的每個部分。

![應用程式總覽](../../../assets/images/app-overview.png)

### 側邊欄導覽

側邊欄分為三個區段：

**總覽**
:   你的主要儀表板：包含圖表與績效指標的投資組合摘要。

**投資組合**
:   - **快照**：瀏覽與管理投資組合快照
    - **資產**：檢視與編輯所有追蹤的資產
    - **類別**：將資產分組並設定目標配置
    - **平台**：管理持有資產的平台或券商

**工具**
:   - **再平衡**：計算達到目標配置所需的交易
    - **匯入 CSV**：從試算表檔案批次匯入資產或現金流
