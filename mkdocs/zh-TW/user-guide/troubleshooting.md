# 疑難排解

使用 AssetFlow 時可能遇到的常見問題及其解決方法。

## 匯率

### 匯率無法載入

- 檢查你的網路連線。
- 在快照詳細檢視中點選**重試**按鈕。
- 匯率從 cdn.jsdelivr.net 取得，請確認你的網路可以連線到此網域。
- 如果你使用防火牆或 VPN，可能會阻擋連線。

### 匯率顯示過時或不正確

- 匯率是針對特定的快照日期取得的。歷史匯率反映的是該日期的實際匯率，而非今天的匯率。
- 如果匯率看起來有誤，可能是因為資料來源使用的是收盤匯率，與你在其他地方看到的盤中匯率可能不同。

---

## CSV 匯入

### 匯入按鈕無法點選

預覽中有驗證錯誤。查找紅色的錯誤標記並修正問題。常見原因包括缺少欄位、重複的資產，或無法解析的數值。

### 匯入後文字出現亂碼

你的 CSV 檔案可能不是以 UTF-8 編碼儲存。請從試算表應用程式重新匯出，選擇 UTF-8 編碼。

### 「重複的資產」錯誤

目標快照中已存在相同名稱和平台的資產。你可以重新命名資產、變更平台，或從 CSV 中移除重複的列。

---

## 備份與還原

### 還原後似乎遺失了資料

還原會以備份內容取代所有目前的資料。如果你有比備份更新的資料但不在備份中，那些資料就會消失。請務必在從舊備份還原之前，先匯出目前資料的備份。

---

## 應用程式鎖定

### Touch ID 無法使用

Touch ID 需要配備 Touch ID 感應器的 Mac（MacBook Pro/Air 含 Touch Bar 或更新款式）。在其他 Mac 上，請改用系統密碼。

### 無法解鎖應用程式

- 試試你的 Mac 系統登入密碼，它永遠可以作為備用驗證方式。
- 如果你最近變更了系統密碼，新密碼應該可以使用。

### 應用程式鎖定得太快

在**設定**（++cmd+comma++）**> 安全性**中調整逾時設定。將「切換應用程式時」和「螢幕鎖定或睡眠時」設為較長的間隔或「永不」。

---

## 一般問題

### 快照很多時應用程式變慢

大型投資組合（有數百項資產橫跨多筆快照）可能會出現這種情況。效能取決於你的 Mac 硬體。

!!! tip

    如果你遇到這裡沒有列出的問題，請嘗試重新啟動 AssetFlow。如果問題持續存在，你可以在專案的 GitHub 頁面上回報。
