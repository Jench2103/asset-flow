# 通知

AssetFlow 可以定期發送本機提醒，讓你不會忘記建立新的快照。提醒功能預設為關閉——你可以在「設定」中自行啟用，並隨時調整頻率或再次關閉。

## 啟用快照提醒

1. 開啟**設定**（++cmd+comma++），前往**通知**區段。
1. 開啟**快照提醒**。
1. 第一次啟用時，macOS 會詢問是否允許 AssetFlow 發送通知。請在系統對話框中點選**允許**。

![通知設定](../../assets/images/settings-notifications.png)

如果你之前曾在**系統設定 → 通知**中拒絕 AssetFlow 的通知，切換開關時會自動還原為關閉，並顯示「通知已停用」的提示，內含**開啟系統設定**按鈕。請於該處啟用 AssetFlow 的通知，再回到 AssetFlow 重試。

## 選擇頻率

啟用後，選擇你希望多久收到一次提醒：

| 頻率         | 設定方式                             |
| ------------ | ------------------------------------ |
| **每天**     | 選擇一個時間                         |
| **每週**     | 選擇星期幾與時間                     |
| **每 2 週**  | 選擇星期幾與時間                     |
| **每月**     | 選擇每月的日期（1–28）與時間         |
| **自訂間隔** | 選擇 2 到 365 天的任意間隔，加上時間 |

每月日期的選項只到 28 號，是為了讓提醒在每個月都能一致觸發——不會在 2 月略過、也不會無預警地落在月底。

自訂間隔模式適合不對齊星期或月份邊界的週期，例如每 10 天建立一次快照。第一次提醒會在你儲存設定後的下一個指定時間發出，之後每 N 天觸發一次。

![自訂間隔通知](../../assets/images/settings-notifications-custom-interval.png)

## 提醒長什麼樣子

提醒觸發時，你會看到一個標題為 **AssetFlow 提醒** 的通知橫幅，內容是「該為您的投資組合建立新的快照了。」橫幅本身不包含任何財務資料，因此即便顯示在鎖定畫面上也是安全的。

橫幅提供兩種操作：

- **點按橫幅**：將 AssetFlow 帶到前景，並在「快照」畫面中開啟「新增快照」面板，讓你選擇日期與建立模式（空白、複製最新一筆，或批次輸入）。
- **明天再提醒**：在 24 小時後安排一次性的提醒，不影響原本的循環排程。

橫幅自動消失後，提醒仍會保留在 macOS 的**通知中心**裡，直到你手動清除。

!!! tip

    如果你希望橫幅停留在螢幕上直到你點擊（而不是幾秒後自動消失），請開啟**系統設定 → 通知 → AssetFlow**，將通知樣式從**橫幅**切換為**提示**。macOS 將此偏好設定為每個 App 的個別設定，AssetFlow 無法替你變更。

## 暫停或停用

- 若只想跳過下一次提醒，請在橫幅上點擊**明天再提醒**。
- 若要完全停用提醒，請在「設定」中關閉**快照提醒**。所有待觸發的提醒會立即被清除。
- 若要調整頻率，直接修改選項即可——AssetFlow 會自動重新排程。

!!! note

    提醒是由 macOS 自己排程的，因此即使 AssetFlow 已關閉也會觸發。排程會在你結束 App、重新啟動 Mac、或讓 Mac 進入睡眠後仍然保留。
