# 安全性

AssetFlow 提供選擇性的應用程式鎖定功能，保護你的財務資料不被未經授權的人存取。啟用後，一個全螢幕的遮罩會隱藏所有投資組合資料，直到你完成驗證。

## 啟用應用程式鎖定

1. 開啟**設定**（++cmd+comma++），前往**安全性**區段。
1. 開啟**需要驗證**。

![安全性設定](../../assets/images/settings-security.png)

## 驗證方式

可用的驗證方式取決於你的 Mac 的功能：

- **Touch ID**：適用於配備 Touch ID 感應器的 MacBook（MacBook Pro/Air 含 Touch Bar 或更新款式）
- **Apple Watch**：適用於已配對 Apple Watch 的情況
- **系統密碼**：在任何 Mac 上都可作為備用方式使用

## 逾時設定

透過兩個獨立的逾時設定控制 AssetFlow 何時自動鎖定：

### 切換應用程式時

切換到其他應用程式後（例如按 ++cmd+tab++），多快自動鎖定。

| 選項       | 行為                                      |
| ---------- | ----------------------------------------- |
| **立即**   | 離開 AssetFlow 的瞬間就鎖定               |
| **1 分鐘** | 離開 1 分鐘後鎖定                         |
| **5 分鐘** | 離開 5 分鐘後鎖定                         |
| **永不**   | 維持解鎖狀態，直到 Mac 進入睡眠或螢幕鎖定 |

### 螢幕鎖定或睡眠時

Mac 螢幕鎖定或進入睡眠後，多快自動鎖定。

| 選項       | 行為                           |
| ---------- | ------------------------------ |
| **立即**   | 螢幕鎖定或睡眠的瞬間就鎖定     |
| **1 分鐘** | 螢幕鎖定或睡眠 1 分鐘後鎖定    |
| **5 分鐘** | 螢幕鎖定或睡眠 5 分鐘後鎖定    |
| **永不**   | 即使經過睡眠週期仍維持解鎖狀態 |

!!! tip

    如果你經常在 AssetFlow 和試算表之間切換更新數值，可以將「切換應用程式時」設為「永不」。

## 鎖定畫面

鎖定時，一個全螢幕的遮罩會完全隱藏所有投資組合資料。若要解鎖：

1. 在鎖定畫面上點選**解鎖**。
1. 使用 Touch ID、Apple Watch 或系統密碼進行驗證。

應用程式在變為使用中狀態時會自動嘗試解鎖，所以在很多情況下，你只需要看一眼 Touch ID 感應器或輕觸 Apple Watch 即可。

![鎖定畫面](../../assets/images/lock-screen.png)

!!! note

    當應用程式處於鎖定狀態時，工具提示與滑鼠懸停效果也會被隱藏，以防止資料透過鎖定遮罩洩漏。
