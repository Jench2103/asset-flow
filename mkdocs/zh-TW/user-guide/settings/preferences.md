# 偏好設定

自訂 AssetFlow 顯示與處理資料的方式。從選單列的 **AssetFlow > 設定** 開啟設定，或按 ++cmd+comma++。

## 主要幣別

選擇用來顯示所有投資組合數值的幣別。當你的投資組合包含不同幣別的資產時，AssetFlow 會使用取得的匯率將所有數值換算為此主要幣別。

從下拉選單中選擇你偏好的幣別。支援的完整幣別清單請參閱[幣別與匯率](../reference/currencies.md)。

## 日期格式

選擇整個應用程式中日期的顯示方式。提供多種格式選項，並附有即時預覽，讓你確切看到日期會如何呈現，方便你挑選最習慣的風格。

## 匯入預設

設定在 CSV 匯入時預先填入的預設平台名稱。如果你經常從同一家券商或平台匯入，這可以省去你每次都要重新輸入相同名稱的麻煩。

## 關於

關於區段顯示：

- **應用程式版本**與建置編號
- **開發者資訊**與授權條款
- **隱私資訊**：
    - 所有資料都儲存在你的 Mac 本機
    - 匯率從 cdn.jsdelivr.net 取得
    - 不會收集或傳輸任何個人資料
- **原始碼**：專案 GitHub 儲存庫的連結

![設定](../../assets/images/settings.png)

!!! tip

    切換主要幣別後，所有投資組合的數值與圖表會立即更新為新的幣別。換算時使用的是匯率資料，因此請確保你的快照已成功取得匯率。
