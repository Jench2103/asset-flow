# 績效指標

AssetFlow 計算多項績效指標，幫助你了解投資表現。如果你曾好奇儀表板上那些數字代表什麼意思，這個頁面會用白話逐一說明。

## 成長率

**是什麼：** 投資組合總值在兩個日期之間的簡單百分比變化。

**計算方式：** 用期末值減去期初值，除以期初值，再乘以 100%，就是你的成長率。

**重要須知：** 成長率包含了存入與提領的影響。如果你在期間內存入了 $10,000，即使投資完全沒有漲跌，投資組合的成長率仍會反映這筆存入。

**最適用場景：** 快速了解你的總財富變化。當你想看到全貌（包含你存入或提領的資金）時很有用。

## 報酬率（修正 Dietz）

**是什麼：** 經過現金流調整的報酬率，排除了存入與提領的影響，呈現純粹的投資表現。

**計算方式：** 修正 Dietz 方法會考慮資金在期間中*何時*被存入或提領，依據實際投入的時間加權計算。月初存入的資金權重高於月底存入的資金。

**為什麼重要：** 如果你在月中存入 $10,000，修正 Dietz 方法不會將這筆存入計為「成長」。它只衡量你原有的資金（以及新存入的資金，從存入時間起算）實際上表現如何。

**最適用場景：** 了解你在單一期間的實際投資表現。當你想知道「我的投資這個月表現如何？」而不被存入或提領的金額所誤導時，看這個數字就對了。

## 累計時間加權報酬率（TWR）

**是什麼：** 將每個快照期間的報酬率串聯起來，呈現你的整體投資績效，完全消除現金流時間點的影響。

**計算方式：** AssetFlow 計算每兩個相鄰快照之間的報酬率（使用修正 Dietz 方法），然後將它們複合計算，就像把每個期間的成長因子相乘。最後的結果呈現的是，假設從未有過現金流進出時的整體投資表現。

**為什麼重要：** 即使你在市場下跌前剛好存入一大筆資金，TWR 衡量的是你的*投資選擇*表現如何，而非你存入資金的時間點。兩個持有相同投資但存款時間不同的人，會看到相同的 TWR。

**最適用場景：** 長期績效追蹤。這是業界評估投資績效的標準方法，也是基金經理人和理財顧問使用的指標。

## CAGR（年化複合成長率）

**是什麼：** 累計 TWR 的年化版本，它告訴你如果成長完全平穩一致，你的投資組合每年成長了多少。

**最適用場景：** 將你的投資組合績效與基準指標進行比較。當有人說「S&P 500 每年報酬 10%」時，那就是 CAGR，現在你可以用同樣的標準來比較自己的投資組合。

---

!!! tip

    如果只看一個數字，建議看**累計 TWR**，它是最公平地衡量你的投資實際表現的指標。

!!! note

    為了讓這些指標準確，請務必在每筆快照中記錄你的現金流（存入與提領）。沒有現金流資料，AssetFlow 就無法區分投資收益與你存入的資金。
