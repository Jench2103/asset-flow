# 再平衡

再平衡工具讓你清楚看到目前的投資組合配置與目標的差距，並建議你需要買進或賣出多少才能回到正軌。

!!! note

    AssetFlow 的再平衡工具是一個唯讀的計算機，僅提供建議，實際的交易需要你自行透過券商執行。

## 運作方式

再平衡工具將你的**目前配置**（根據最新快照）與你在各類別上設定的**目標配置**進行比較。它會計算差距，並告訴你每個類別需要買進或賣出多少才能達到目標。

## 解讀再平衡表格

頂部會顯示你的**投資組合總值**。

下方的表格分為三個區段：

1. **有目標的類別**：主要的再平衡檢視，顯示已設定目標配置的類別。
1. **未設定目標**：存在但尚未設定目標配置的類別。
1. **未分類**：尚未指定到任何類別的資產。

表格中每一列包含：

| 欄位       | 說明                     |
| ---------- | ------------------------ |
| **類別**   | 類別名稱                 |
| **目前值** | 此類別中資產的總值       |
| **目前 %** | 此類別佔投資組合的比例   |
| **目標 %** | 你為此類別設定的理想配置 |
| **差距**   | 目前配置與目標的落差     |
| **操作**   | 達到目標所需採取的行動   |

操作以顏色標示，方便快速瀏覽：

- **買進**（綠色）：你需要買進的金額，才能達到目標。
- **賣出**（紅色）：你需要賣出的金額，才能降到目標。
- **無需操作**（灰色）：你已經在目標上了，做得好！

## 建議操作

檢視底部有一個**建議操作**摘要，以文字描述說明讓投資組合恢復平衡所需的主要操作。

## 前置條件

要使用再平衡工具，你需要：

- 至少一筆包含資產的快照。
- 已設定目標配置的類別。
- 已將資產指定到類別。

!!! tip

    先為你的類別設定目標配置（例如：股票 60%、債券 30%、現金 10%），再平衡工具就會自動幫你計算需要做什麼。
