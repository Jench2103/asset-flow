# 平台

平台代表你的資產存放在哪裡，例如券商帳戶、交易所、銀行或其他保管機構。它幫助你追蹤哪些資產在哪個平台上，讓你隨時知道你的資金在哪裡。

大多數時候你不需要手動建立平台。平台通常在你匯入 CSV 檔案或為資產指定平台時自動建立。

## 平台列表

平台檢視的左側列出所有平台，可排序顯示。

- **拖曳排序**：依你喜歡的方式排列平台順序。
- 每一列顯示**平台名稱**、**總值**與**資產數量**。
- 在任一平台上**按右鍵**可重新命名。

![平台列表](../../assets/images/platform-list.png)

## 平台詳細檢視

從列表中選取一個平台，即可在右側查看其詳細資訊。

- **名稱**：顯示在頂部。在列表中的平台上按右鍵可重新命名。
- **平台上的資產**：一個表格列出此平台持有的所有資產，以及每項資產的名稱與類別。
- **價值歷史**：一張折線圖顯示此平台的總值隨時間的變化。使用時間範圍選擇器可聚焦特定期間，將游標懸停在任一資料點上可查看確切的日期與數值。

![平台詳細檢視](../../assets/images/platform-detail.png)

!!! note

    你不需要手動建立平台。當你匯入 CSV 資料或為資產指定平台時，平台會自動建立。
