# 儀表板

儀表板是你開啟 AssetFlow 時看到的第一個畫面，也是你的主畫面（位於側邊欄的**總覽 > 儀表板**）。它讓你一眼掌握整個投資組合的概況。

## 摘要卡片

儀表板頂部有一組摘要卡片：

- **投資組合總值**：最顯眼的主要卡片，顯示你的投資組合目前的總價值。
- **最新快照日期**：你最近一筆快照的日期。
- **資產數量**：你的投資組合中有多少項資產。
- **累計 TWR**：你的歷史時間加權報酬率（業界衡量投資績效的標準方法）。
- **CAGR**：你的年化複合成長率，方便你進行逐年比較。

## 期間績效卡片

摘要下方，你可以選擇 **1M**、**3M** 或 **1Y** 等時間區間，查看該期間的投資組合表現。每張績效卡片顯示：

- **成長率**：投資組合總值的簡單百分比變化。這個數字包含了存入與提領的影響。
- **報酬率**：使用修正 Dietz 方法計算的現金流調整後報酬率。這個指標排除了資金進出的影響，呈現你真正的投資表現。
- **日期範圍副標題**，標示涵蓋的確切期間。

![績效卡片](../../assets/images/dashboard-performance.png)

!!! tip

    不確定成長率與報酬率有什麼差別嗎？前往[績效指標](performance-metrics.md)頁面查看白話說明。

## 圖表

儀表板包含多個互動式圖表，幫助你視覺化投資組合：

- **類別配置圓餅圖**：檢視你的投資組合在各類別間的分布。點選某個類別的扇形區域，可直接跳至該類別的詳細頁面。
- **投資組合價值折線圖**：追蹤你的投資組合總值隨時間的變化。點選任一資料點可跳至對應的快照。
- **累計 TWR 折線圖**：顯示你的時間加權報酬率隨時間的走勢，讓你了解投資績效的趨勢。
- **類別價值折線圖**：追蹤每個類別的價值隨時間的變化，方便你找出投資組合中成長最快的部分。

所有圖表都支援**滑鼠懸停提示**，只要將游標移到任一資料點上，就能看到確切的日期與數值。

![儀表板總覽](../../assets/images/app-overview.png)

## 最近快照

儀表板底部列出你最近的幾筆快照。每一列顯示快照日期、總值與資產數量。點選任一列可直接跳至該快照的詳細檢視。

## 空白狀態

如果你尚未建立任何快照，儀表板會顯示引導提示，幫助你開始使用。你可以選擇匯入 CSV 檔案或手動建立第一筆快照。
