# 類別

類別讓你依類型將資產分組，例如股票、債券、不動產、現金，或任何適合你投資組合的分類方式。每個類別可以設定**目標配置**，協助你進行再平衡。

## 類別列表

類別檢視的左側列出所有類別。你可以**拖曳排序**，這裡設定的順序會套用到整個應用程式。

每一列顯示：

- 類別名稱
- 目標配置 %
- 目前配置 %
- 目前價值
- 資產數量
- 偏差指示器（**警告圖示**），當目前配置與目標的差距超過 5% 時會出現

### 橫幅

列表頂部可能出現兩個資訊橫幅：

- **目標配置加總警告**：當你的目標配置總和超過 100%，或部分類別缺少目標設定時顯示。幫助你發現設定上的問題。
- **顯著偏差提示**：當一個或多個類別與目標有明顯偏差時顯示，並附上前往[再平衡](rebalancing.md)工具的連結。

在類別上按右鍵可刪除（僅當沒有資產被指定到該類別時才可刪除）。

![類別列表](../../assets/images/category-list.png)

## 建立類別

點選工具列的 **+** 按鈕建立新類別。你需要輸入：

- **類別名稱**：為該群組取一個描述性的名稱（例如「股票」、「固定收益」、「另類投資」）。
- **目標配置 %**：選填，介於 0 到 100 之間。這是你希望此類別在投資組合中佔的百分比。

## 類別詳細檢視

選取一個類別，即可在右側查看其詳細資訊。

### 屬性

- **名稱**：可直接在詳細檢視中編輯。
- **目標配置 %**：可編輯，範圍同樣是 0 到 100。

### 類別中的資產

一個表格列出所有歸屬於此類別的資產，包含每項資產的名稱與平台。

### 價值歷史

一張折線圖顯示此類別的總值隨時間的變化。使用時間範圍選擇器可聚焦特定期間。

### 配置歷史

一張折線圖顯示此類別的配置百分比隨時間的變化。圖上會疊加一條**目標線**供你參考，讓你一眼看出實際配置與目標的差距。

![類別詳細檢視](../../assets/images/category-detail.png)

## 刪除類別

類別只有在**沒有任何資產被指定到它**時才能刪除。如果類別中仍有資產，你需要先將那些資產重新指定到其他類別，然後才能刪除這個空類別。

!!! tip

    使用[再平衡](rebalancing.md)工具可以一目了然地看出哪些類別超過或低於目標配置。它是你決定下一步投資方向的最快方式。
