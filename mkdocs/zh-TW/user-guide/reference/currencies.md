# 幣別與匯率

AssetFlow 支援多種幣別，並自動取得匯率，將所有數值換算為你的主要顯示幣別。

## 運作方式

- 每項資產可以用自己的幣別追蹤（例如美股用 USD，台股用 TWD）。
- 系統會自動取得每筆快照日期的匯率。
- 所有數值會換算為你的主要幣別顯示。你可以在**設定**（++cmd+comma++）**> 偏好設定**中變更主要幣別。
- **資料來源**：匯率從 cdn.jsdelivr.net 取得，提供開放原始碼的匯率資料。

## 支援的幣別

AssetFlow 內建一組常用幣別清單。首次啟動時（以及之後定期更新），應用程式會從匯率 API 取得擴充的幣別清單，並快取在本機供離線使用。

### 完整幣別清單

首次成功連線後，AssetFlow 會從[匯率 API](https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies.json) 取得完整的可用幣別清單，並快取在本機。這包含了數百種法定貨幣與加密貨幣，遠超上方的內建清單。

你可以在瀏覽器中開啟 [API 端點](https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies.json)查看完整的支援幣別代碼清單。

## 自動取得行為

匯率在以下幾種情況下會自動取得：

- **建立新快照時**：當你建立的快照包含多種幣別的資產時，該日期的匯率會立即取得。
- **啟動應用程式時**：任何現有快照缺少的匯率會在背景中取得。
- **還原備份後**：從備份還原時，系統會重新取得所有快照的匯率。

## 無法取得匯率時

如果你處於離線狀態或遇到網路問題：

- 快照詳細檢視中會顯示**重試**按鈕，讓你稍後再試。
- 數值會以原始幣別顯示，不進行換算。
- 應用程式的其他功能完全可以離線使用，匯率是唯一需要網路的功能。

!!! note

    AssetFlow 會將取得的匯率儲存在本機。一旦某筆快照的匯率已成功取得，即使離線也不需要重新取得。
