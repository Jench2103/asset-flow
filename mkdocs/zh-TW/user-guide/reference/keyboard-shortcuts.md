# 鍵盤快捷鍵

AssetFlow 鍵盤快捷鍵速查表。

| 快捷鍵        | 功能                                     |
| ------------- | ---------------------------------------- |
| ++cmd+n++     | 建立新快照                               |
| ++cmd+i++     | 開啟 CSV 匯入                            |
| ++cmd+comma++ | 開啟設定                                 |
| ++enter++     | 跳到下一列的新值欄位（在批次輸入中使用） |

!!! note

    未來的更新可能會新增更多鍵盤快捷鍵。
